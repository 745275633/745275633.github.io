开始之前

什么是Prolog

Prolog 是 Programming in Logic 的缩写，意思就是使用逻辑的语言编写程序。Prolog 不是很难，相反，比其他的编程语言，例如c、basic等等语言， Prolog 是很容易理解的语言。如果你从来没有接触过编程，那么你将很容易掌握 Prolog。如果你能熟练的使用一般的编程语言，那么你需要改变一下思维方式，否则是比较难掌握 Prolog 的。

举一个很著名的例子

  • 张学友爱王菲
  • 张学友爱周慧敏
  • 王菲爱谢廷峰
  • 周慧敏爱张学友
  • 谢廷峰爱王菲
  • 谢廷峰爱周慧敏
  • 刘德华爱周慧敏

那么怎么表示出这些事实呢？

用 Prolog 表示就是

爱(张学友, 王菲).

这就被称作事实

注意：

  1. 所有符号都要使用半角的（英文符号）也就是(),.，而不是（），。
  2. 最后的.一定不能没，这个.就相当于 c/c++ 中的;，表示一条语句结束了
  3. 这些词汇对电脑来说没有意义，如果你愿意，你也可以写成xxx(zxy, wf).，只要你自己明白
  4. 有些教程会说 Prolog 不支持中文，事实上是可以的，这条语句在 swi-prolog 中通过，在 GNU Prolog（在swi-prolog中也可以） 中需要在中文外面加单引号’爱‘(’张学友‘, ’王菲‘).，也必须是半角的。但是，gprolog 不支持输入中文，必须要使用重定向从文件中才能输入，比较麻烦。

例子中的事实翻译成 Prolog 就是

'爱'('张学友', '王菲').
'爱'('张学友', '周慧敏').
'爱'('王菲', '谢廷峰').
'爱'('周慧敏', '张学友').
'爱'('谢廷峰', '王菲').
'爱'('谢廷峰', '周慧敏').
'爱'('刘德华', '周慧敏').

那他们中，有哪些人是情侣呢？

电脑不知道什么是情侣，所以要告诉它情侣的规则

情侣 (甲, 乙) :- 爱 (甲, 乙), 爱 (乙, 甲).

:-表示如果，这句话翻译成人类的语言就是如果甲爱乙，而且乙爱甲。那么甲和乙就是情侣

全部用英语表示（ gprolog 的中文支持的不太好）就是：

love(zhangxueyou, wangfei).
love(zhangxueyou, zouhuimin).
love(wangfei, xietingfeng).
love(zouhuimin, zhangxueyou).
love(xietingfeng, wangfei).
love(xietingfeng, zouhuimin).
love(liudehua, zouhuimin).
lovers(X,Y):-love(X,Y),love(Y,X).

在解释器（gprolog 或 swipl ，就是在第一课测试的那个）中输入[user].然后敲两下回车，输入这段文字之后，敲一下回车，然后，windows 输入 <Ctrl-Z> ，其他系统输入 <Ctrl-D> 反正就是输入EOF。

现在，解释器已经知道谁爱谁和情侣是什么了，解释器的提示符是?-，你可以认为和shell的$差不多，现在可以问它。如：

？- love(zhangxueyou, wangfei).

表示张学友爱王菲吗，swi 会告诉你

true

你再敲一下回车会出现一个. ，而GNU会出现

true ?

那个问号不是不确定，而是问你还要继续吗，在这里直接敲回车就会有

yes

表示找到了。

在 Prolog 中，大写字母开头的叫做变量，解释器会求出变量的值。例如：

?- love(zhangxueyou,X).

这句话是说，张学友喜欢哪些人，在GNU中，会显示

X = wangfei ?

你直接输入回车就结束了，输入;可以求出下一个（组）解：

X = wangfei ? ;

X = zouhuimin

yes

也可以输入a输出全部解。

在swi中

?- love(zhangxueyou,X).
X = wangfei ;
X = zouhuimin.

不能输出全部解，只能一个一个输出。

现在我们来试一下规则

lovers(wangfei,Y).
Y = xietingfeng

解释器告诉我们，王菲的情侣是谢霆锋

我们再来问一下有哪些情侣

?- lovers(X, Y).
X = zhangxueyou,
Y = zouhuimin ;

Y = xietinfeng ;
X = zouhuimin,
Y = zhangxueyou ;
X = xietinfeng,
Y = wanfei ;
false.

最后那个false，表示找不到更多了

GNU则是

X = zhangxueyou
Y = zouhuimin ? a

X = wangfei
Y = xietingfeng

X = zouhuimin
Y = zhangxueyou

X = xietingfeng
Y = wangfei

no

GNU 用的是 yes 和 no ，而 swi 用的是 true 和 false。

因为 Prolog 是区分顺序的 lovers(zhangxueyou,zouhuimin) 和 lovers(zouhuimin,zhangxueyou) 是不一样的。所以每对情侣被输出了两次。
