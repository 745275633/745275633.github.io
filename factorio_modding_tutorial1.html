第1章 - 建立

现在开始，首先你需要找到你的 Factorio 安装文件夹， 这页 可以告诉你如何找到。在全新的 Factorio 中，如果没有 MOD 文件夹，你需要先启动一场游戏。

开始制作mod时我们需要一个想法：我想制作一个能在游戏中使用轰炸机的MOD。为什么？因为只需简单地从虫巢上方飞过就能使用炸弹把他们炸个稀巴烂，这是多么惬意 :) 而且，没有类似的MOD。所以，和普通的 Factorio mod 比，它包含了更多的步骤和复杂的技术才能制作。

1) 在MODS的文件夹中创建一个名为 BomberTutorial_0.1.1 的新文件夹，后面的 0.1.1 需要和 info.json 中 version 的值一样。

2) 在 BomberTutorial_0.1.1 文件夹中，创建3个文件：

  • info.json （有关 mod、作者、版本 等信息）
  • data.lua （包含其他 .lua 文件 的加载定义）
  • control.lua （希望 mod 执行的操作)

3) 用你喜欢的文本编辑器，编辑 info.json 文件，（或直接 复制/粘贴）：

{
  "name": "BomberTutorial",
  "version": "0.1.1",
  "title": "FreeER's Step by Step Bomber Mod",
  "author": "FreeER",
  "contact": "http://www.factorioforums.com/forum/viewtopic.php?f=25&t=4570",
  "homepage": "http://www.factorioforums.com/wiki/index.php?title=Modding_Tutorial",
  "description": "Homemade Mod, added a bomber"
}
  • name （名字）
  • version （版本）
  • title （在游戏中 mod 界面的菜单中显示的标题）
  • author （作者）
  • contact （联系方式）
  • homepage （主页）
  • description （介绍）

4) 在bombertutorial文件夹中创建一个名为“原型”文件夹。这个文件夹用来放置定义所有我们希望在我们的国防部使用的新物品的文件，配方和科技。在接下来的章节中，我们将创建一个文件在这个文件夹中。

现在，我们（在技术上）已经有了一个MOD！虽然是个空架子，但是已经是一个mod。

5.) 保存所有文件，并运行Factorio。在主菜单中，点击MODS按钮，然后你将看到我们的MOD！但是由于MOD是个空架子，我们现在可以关闭Factorio了。
